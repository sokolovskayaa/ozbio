# Тестирование MVP планировщика

Зафиксированная версия (май 2026): Spring Boot, пакетное планирование 24/7, переналадка, Thymeleaf UI. **Без** смен, overlap, переплана и симуляции времени.

## Запуск

```bash
mvn clean test
```

## Демо-скрипты (ручная проверка)

| Скрипт | Назначение |
|--------|------------|
| `./scripts/reset-demo-data.sh` | Сброс `data/schedule.json` из example (сервер **остановлен**) |
| `./scripts/demo.sh` | POST заказов 0142 + 0148, проверки, сохранение HTML |
| `./scripts/prep-director-demo.sh` | Проверка готовности к встрече |
Документ ручных сценариев: [согласование-с-директором.md](согласование-с-директором.md).
